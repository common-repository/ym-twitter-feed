This plugin can be used to embed twitter feed to your website.

== Description ==

You can use this plugin to show twitter feeds to your wordpress website. You have a widget control panel that will allow you to configure your twitter username, theme, border, bg color, data link color, width of the screen to display the feed and number of feeds that you want to display. You can drag and drop this widget to any sidebars.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip and Upload `ym-twitter-feed.zip` to the `/wp-content/plugins/` directory or Upload the zip from the Wordpress Admin > plugin > Add Plugin > Upload Plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance > Widgets and drag the widget to desired sidebar and configure is as necessary.
